# TP Dash — Dashboard de ventes 
Ce dépôt contient un TP complet destiné à découvrir la bibliothèque Dash (Plotly) dans le cadre du cours d’IHM. L’objectif est de construire progressivement un dashboard interactif, en 7 étapes, jusqu’à un rendu final stylé et professionnel. 
La correction du TP se trouve dans la branche TP.


# I - Installation 

1 - Créer un environnement virtuel 
python -m venv venv 

2 - Pour activer 
Sous Linux ou macOS : source venv/bin/activate 
Sous Windows : venv\Scripts\activate 

3 - Installer les dépendances : 
pip install dash dash-bootstrap-components plotly pandas numpy 


# II - Objectifs pédagogiques 

Ce TP permet d’apprendre à : 
- Créer une application web interactive sans JS, uniquement Python
- Manipuler les composants Dash (html, dcc)
- Utiliser Plotly Express pour produire des visualisations
- Utiliser Bootstrap Components pour structurer une page
- Comprendre et écrire des callbacks réactifs
- Organiser un projet Dash dans une structure propre
- Améliorer l’esthétique d’une interface
